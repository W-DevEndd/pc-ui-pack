
# PC-UI-Pack
is a open source UI pack for Mimecraft Bedrock Edition to make the Mobile-UI like Desktop.

# Build
## Requirement
- Enviroment: Unix-like / Unix-base system (WSL also)
- Building Tools: nodejs, make, python
- Package Manager: yarn, pip

## Procedure
Initialization the project (create venv and install dependencies)
```
make init
```

Build the UI pack
```
make
```

## Configs
- ``asakiyuki.config.cjs``: Config the pack manifest, compiler and auto imstaller,...
- ``.is_dev_version.json`` (After initization): Toggle of dev mode or release
